HTML

HTML stands for Hyper TextMarkup Language.

HTML Tags

Tags are keywords that represent elements in HTML and they are surrounded by angle brackets.

HTML Attributes

Attributes come in the form of name/value pair and they provide additional information about the element.

HTML Elements

Elements are everything form the start tag to the end tag.
